The default credential to use when performing operations on this clusterName. If not specified, credentials identifiers may be specified directly in the DSL.

openshift.withCluster( 'my-clusterName', 'my-credentials-id' ) { ... }

openshift.withCluster( 'my-clusterName' ) { openshift.doAs( 'my-credentials-id' ) { ... } }

In the above examples, a Jenkins credential identifier is specified. Alternatively, an OAuth token value may be specified directly in these locations.
